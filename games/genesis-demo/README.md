# LWX Genesis Demo

A tiny original Sega Genesis / Mega Drive game for the LibretroWebXR project's
playable test content. Move a 16x16 sprite around a tiled, two-tone background
with the D-pad; press **A / B / C** to recolour the sprite (gold / green / blue).

This exists to prove the Genesis path end-to-end with content we own and can
ship under a clean license.

## What is "ours"

- `src/main.c` — all the game logic (CC0, hand-written). Uses only SGDK's
  documented API (`VDP_*`, `JOY_*`, `PAL_*`, `SYS_doVBlankProcess`). All artwork
  is defined inline as tile data, so the build needs no PNG assets and no
  rescomp step.

## What is frozen boilerplate (do not hand-edit the layout)

- `src/boot/sega.s` — boot code, copied verbatim from SGDK's known-good template.
- `src/boot/rom_head.c` — the ROM header **as a C struct** generated by SGDK.
  Only the title / copyright text fields were filled in (the documented,
  supported edit). The header byte layout and the m68k vector table are never
  hand-written.

## Build

Requires [SGDK 2.11](https://github.com/Stephane-D/SGDK/releases/tag/v2.11)
extracted to `C:\sgdk` (zip/7z extract only — no installer) and Java on `PATH`
(used by SGDK's `rescomp` / `sizebnd`). Then, from the repo root:

```
node scripts/make-genesis-demo.mjs
```

That runs SGDK's `make -f makefile.gen` (its bundled gcc + make), producing
`out/rom.bin`, and copies it to
`public/roms/freeware/lwx-genesis-demo.md`.

Equivalent manual build, run from this directory:

```
set GDK=C:\sgdk
C:\sgdk\bin\make.exe -f C:\sgdk\makefile.gen
```

Output is a 131072-byte (128 KB) ROM padded/aligned by SGDK's `sizebnd`, with
the `SEGA MEGA DRIVE` signature at offset `0x100`. Runs on the `genesis_plus_gx`
core (system key `md`).

## Controls

- **D-pad** — move the sprite (clamped to the screen edges).
- **A** — gold, **B** — green, **C** — blue.
